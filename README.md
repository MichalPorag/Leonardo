Leonardo Docs 
=============

## Compile

`cd` into the project folder

```bash
npm run compile
```

run local server and navigate to `index.html`

## Auto Recompile

To automatically recompile the files on each change run

```bash
npm run compile:watch
```

run local server and navigate to `index.html`.

Note that this does not auto-refreshes the browser.
